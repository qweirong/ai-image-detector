# AI图像检测器

基于 7 维度综合分析的 AI 生成图像检测工具。

## 检测维度

1. **频域分析** - FFT频谱特征
2. **噪声分析** - ELA误差和噪声一致性
3. **颜色分析** - 颜色分布统计
4. **纹理分析** - LBP和边缘复杂度
5. **元数据分析** - EXIF信息检测
6. **边缘分析** - 边缘统计特征
7. **CLIP内容分析** - 基于CLIP模型的语义内容识别

## 使用方法

上传一张图片，点击"开始检测"即可查看结果。
